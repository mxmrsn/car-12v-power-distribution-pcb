# VW GTI 12V Distribution PCB

Simple 12V distribution block for three accessories (dashcam, phone charger, boost gauge).

## Contents
- `vw-gti-12v-distribution.kicad_sch`
- `vw-gti-12v-distribution.kicad_pcb`
- `PicoSpoX.pretty/PicoSpoX_1x02_P3.00mm_THT.kicad_mod`

## Notes
- Footprint is a generic 2-pin PicoSpoX-style 3.00mm pitch THT footprint. Verify against your exact Molex part.
- PCB uses 1.2 mm traces for both +12V and GND. Adjust if you want thicker copper or different current headroom.
- Board outline is 90 mm x 75 mm. Resize as needed to fit near the fuse box.
